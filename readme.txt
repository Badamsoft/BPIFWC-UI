Easy CSV product importer for WooCommerce.

== Description ==
BadamSoft Product Importer for WooCommerce allows you to import products from CSV files into WooCommerce.

The free version focuses on fast and reliable CSV imports.

Source code for the free admin UI: https://github.com/Badamsoft/BPIFWC-UI
Build instructions: See the repository README for npm/Vite build steps.

= Free features =
* Import products from CSV files
* Basic field mapping
* Manual imports from the WordPress admin

== External services ==
This plugin does not send site or product data to external services during imports in the free version.

The plugin includes optional documentation and support links to badamsoft.com that open only when an administrator clicks them in the WordPress dashboard.
- Data sent: Only the click request initiated by the administrator's browser.
- Purpose: To open documentation or support pages selected by the administrator.
- Service Terms: https://badamsoft.com/terms
- Privacy Policy: https://badamsoft.com/privacy-policy

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/badamsoft-product-importer-for-woocommerce` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Does it work with WooCommerce? =
Yes. WooCommerce must be installed and active.
